# Sorting Algorithms

<h2>This is the JAVA GUI application that exhibit how the sorting algorithms below work and their runtime complexity:</h2>

* Insertion Sort
* Merge Sort
* Quick Sort
* Heap Sort

<h3>Instructions</h3>
Program is launched by the application.jar file
